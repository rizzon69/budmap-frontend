📋 Quick Start Steps

1 Install PostgreSQL

Download and install PostgreSQL for Windows:

Download PostgreSQL
Important: Remember the password you set for the 'postgres' user during installation!

2 Create Database

Open Command Prompt and run:

psql -U postgres CREATE DATABASE budmap; \q

3 Configure .env File

Edit Backend\.env and update your PostgreSQL password:

DB_HOST=localhost DB_PORT=5432 DB_NAME=budmap DB_USER=postgres DB_PASSWORD=your_postgres_password_here

4 Run Setup

Easy Way (Recommended):

Double-click the setup-database.bat file in the project root


Manual Way:

cd C:\fyp\Backend npm install npm run db:test npm run migrate

5 Verify Setup

Test the connection:

npm run db:test

You should see all tables listed and data counts.
