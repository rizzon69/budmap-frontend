STEP 1: Install PostgreSQL
---------------------------
Download: https://www.postgresql.org/download/windows/
IMPORTANT: Remember the password you set for 'postgres' user!

STEP 2: Create Database
---------------------------
Open Command Prompt and run:

    psql -U postgres
    CREATE DATABASE budmap;
    \q

STEP 3: Configure Password
---------------------------
Edit this file: Backend\.env
Change this line: DB_PASSWORD=your_postgres_password_here

STEP 4: Run Setup
---------------------------
EASY WAY:
    Double-click: setup-database.bat

MANUAL WAY:
    cd Backend
    npm install
    npm run db:test
    npm run migrate

STEP 5: Verify
---------------------------
Run this command:
    npm run db:test

You should see "✅ Successfully connected" and a list of tables.

=================================================================================
   🐛 COMMON PROBLEMS & SOLUTIONS
=================================================================================

Problem: "ECONNREFUSED"
Solution: PostgreSQL is not running
  → Open Services (Win+R → services.msc)
  → Find "postgresql-x64-XX"
  → Start the service

Problem: "Database does not exist"
Solution: Create the database first
  → Run: psql -U postgres -c "CREATE DATABASE budmap;"

Problem: "Authentication failed"
Solution: Wrong password in .env file
  → Check Backend\.env
  → Make sure DB_PASSWORD matches your PostgreSQL password

Problem: Need to reset everything
Solution: Drop and recreate database
  → psql -U postgres
  → DROP DATABASE budmap;
  → CREATE DATABASE budmap;
  → \q
  → npm run migrate
