Introduction

Traditional budget management in Nepal's SMEs, NGOs, and educational institutions relies on manual methods — spreadsheets and handwritten records — that are time-consuming, error-prone, and lack transparency.

BudMap is a web-based budget allocation and management application providing a centralised digital platform combining real-time dashboards, multi-level approval workflows, predictive analytics, internal messaging, and automated reporting.

Built using React 18, Node.js, Express, and MongoDB Atlas with Google Gemini AI for intelligent forecasting and Google OAuth for secure authentication, BudMap helps organisations make data-driven decisions and achieve long-term financial sustainability.

Aims and Objectives
Aim:

To design and develop a secure, role-based web application enabling SMEs, NGOs, and educational institutions in Nepal to efficiently manage, track, and analyse budgets through a centralised, data-driven platform with AI-powered forecasting.

Objectives:
  • Implement JWT-based authentication with email verification, Google OAuth, and role-based access control for Admin, Finance Officer, Department Head, and Viewer roles.
  • Enable creation, allocation, and tracking of budgets by fiscal year and department with multi-level approval workflows (draft → active → closed).
  • Build income and expense management with a multi-level approval system that auto-updates budget utilization and triggers email alerts at 80% usage.
  • Develop real-time dashboards and deterministic linear regression forecasting integrated with Google Gemini AI for intelligent budget insights.
  • Implement direct messaging with file attachments (images, PDF, Word, MP4 up to 20MB), read receipts, and threaded conversations.
  • Generate financial, department-wise, budget performance, and expense trend reports with PDF export functionality.
Academic Questions
  • How can role-based access control enhance transparency and security in a web-based budget management system for SMEs, NGOs, and educational institutions in Nepal?
  • How can AI-powered forecasting using linear regression and Google Gemini improve financial planning accuracy for organisations with limited resources?
  • How can real-time analytics and automated reporting reduce administrative burden and support data-driven financial decision-making in Nepalese organisations?

Evaluation/Reflection

BudMap successfully implements all core objectives. Role-based access control, budget tracking, multi-level transaction approval, and AI-powered forecasting are fully operational through a React 18 and Node.js stack. JWT authentication, real-time dashboards, analytics, internal messaging, and report generation all work as intended.

From a professionalism perspective, ethical considerations were central — data privacy through role-based transparency, responsible AI predictions with clear uncertainty labels, and user accountability via audit trails. Legal compliance was addressed through bcrypt hashing, MongoDB-backed token storage, and structured documentation aligned with ACM and IEEE standards.

The social impact is significant: BudMap reduces administrative burden for Nepal's SMEs and NGOs, promotes data-driven financial governance, and improves transparency in fund allocation. Gaps remain in WebSocket real-time notifications and two-factor authentication — planned for Phase 2.

Future Plans
  • Enhanced Email Notifications: Extend Gmail SMTP integration for all user-facing events including budget approvals and spending alerts.
  • Excel Export: Generate downloadable reports in Excel format in addition to existing PDF export functionality.
  • Two-Factor Authentication: Add OTP-based 2FA for enhanced account security across all user roles.
  • Mobile Application: Develop a React Native mobile app for on-the-go budget monitoring and transaction approvals.
  • WebSocket Notifications: Implement real-time Socket.io alerts for budget approvals, transaction updates, and threshold warnings.
  • Bank API Integration: Connect with Nepalese banking APIs for automated transaction syncing and real-time financial data import.
