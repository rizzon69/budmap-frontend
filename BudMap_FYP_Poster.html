Introduction

BudMap is a web-based budget allocation and management application designed specifically for SMEs, NGOs, and educational institutions in Nepal. Many of these organisations still rely on manual, spreadsheet-based, or outdated budgeting methods that are time-consuming, error-prone, and lack transparency.

BudMap provides a centralised digital platform for financial management — combining real-time dashboards, multi-level approval workflows, predictive analytics, internal messaging, and automated reporting. Built using React 18, Node.js, Express, and MongoDB Atlas, it uses Google Gemini AI for intelligent budget forecasting and Google OAuth for secure authentication.

The platform addresses critical challenges: lack of transparency, poor resource allocation, absence of role-based collaboration, and inability to generate timely financial insights — helping organisations make data-driven decisions and achieve long-term financial sustainability.

Aims

Aim: To design and develop a secure, role-based web application that enables SMEs, NGOs, and educational institutions in Nepal to efficiently manage, track, and analyse budgets through a centralised, data-driven platform with AI-powered forecasting.

Objectives:
  • User Registration & Auth: Implement JWT-based authentication with email verification, Google OAuth, and role-based access control for Admin, Finance Officer, Department Head, and Viewer roles.
  • Budget Management: Enable creation, allocation, and tracking of budgets by fiscal year and department with multi-level approval workflows.
  • Transaction Tracking: Build income and expense management with a multi-level approval system that auto-updates budget utilization on approval.
  • Analytics & AI Forecasting: Develop real-time dashboards and deterministic linear regression forecasting, integrated with Google Gemini AI for intelligent budget insights.
  • Internal Messaging: Implement direct messaging with file attachments (images, PDF, Word, MP4), read receipts, and conversation threads.
  • Reporting: Generate financial, department-wise, budget performance, and expense trend reports with PDF export functionality.
  • Security & Professionalism: Apply ethical, legal, and social considerations — including RBAC, data encryption, audit trails, and secure password management throughout development.
Academic Questions
How can the integration of role-based access control, AI-powered budget forecasting, and real-time analytics enhance the performance, transparency, and usability of a web-based budget management system for SMEs, NGOs, and educational institutions in Nepal — while addressing ethical, legal, and social responsibilities in its development?

Evaluation/Reflection

BudMap successfully integrates role-based access control, budget tracking, multi-level transaction approval workflows, and AI-powered forecasting through a clean React and Node.js stack. Core features including authentication, dashboards, analytics, and messaging are fully operational.

From a professionalism perspective, ethical considerations were central to the design — including data privacy, role-based transparency, and responsible use of AI predictions. Legal compliance was addressed through JWT security, bcrypt password hashing, and audit trail logging. The social impact is significant: BudMap reduces administrative burden for Nepal's SMEs and NGOs while promoting data-driven financial governance.

Gaps remain in real SMTP email delivery, file upload UI, and PDF generation — planned for Phase 2. Building BudMap deepened understanding of full-stack development, RESTful API design, MongoDB, and Google Gemini AI integration. Overcoming challenges with JWT security and linear regression forecasting provided valuable real-world engineering experience.

Future Plans
  • Real Email Integration: Connect Gmail SMTP via Nodemailer for live email verification, password reset, and budget alert notifications.
  • PDF & Excel Export: Generate downloadable financial and department-wise reports in PDF and Excel format directly from the dashboard.
  • Two-Factor Authentication: Add 2FA support for enhanced account security across all user roles.
  • Mobile Application: Develop a React Native mobile app for on-the-go budget monitoring, approvals, and real-time notifications.
  • WebSocket Notifications: Implement real-time Socket.io alerts for budget approvals, transaction updates, and budget limit warnings.
  • Bank API Integration: Connect with Nepalese banking APIs for automated transaction syncing and real-time financial data import.
